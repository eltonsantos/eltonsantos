# Elton Santos 👨‍💻

### Hi there, welcome to my profile! 👋

My name is Elton Santos from Brazil (🇧🇷). I work with development since 2011 and I'm currently a Systems Analyst at [Intrapoint](https://www.intrapoint.com/).
